How to use:

  1. Download zip or tar file.
  2. Unzip file and place the folder somewhere convenient.
  3. Go to: chrome://extensions.
  4. Make sure developer mode is on.
  5. Click "Load unpacked extension" and select the Bookmarx folder.
  6. Make sure the extension is enabled.
  7. Open a new tab :)
This entension will not be on the Chrome WebStore until it is further developed and properly working to my content.
